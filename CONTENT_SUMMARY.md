## 🚀 Getting Started

### Quick Start

```bash
# Clone repository
git clone https://github.com/pkurri/crucible.git
cd crucible

# Browse templates
ls templates/

# Use a template
cd templates/031-ai-code-reviewer
npm install
npm run dev
```

### Explore Content

```bash
# View all templates
cat templates/CATALOG.md

# View all skills
cat skills/CATALOG.md

# Read guides
cat docs/templates-guide.md
cat docs/skills-guide.md
```
