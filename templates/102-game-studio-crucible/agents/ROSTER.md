# 🎭 The Studio Roster — 48 Specialized Agents

The 48-agent hierarchy provides extreme specialization for game development.

## Tier 1 — The Directors (3 Agents)
- **Creative Director**: High-concept, vision, and "Fun" factor.
- **Technical Director**: Architecture, engine choice, performance standards.
- **Producer**: Scheduling, scope, resource allocation, sprint-planning.

## Tier 2 — The Department Leads (8 Agents)
- **Game Design Lead**: Game systems, GDD ownership, player psychology.
- **Lead Programmer**: Code standards, PR reviews, system architecture.
- **Art Director**: Style guides, color theory, asset pipeline.
- **Audio Director**: Soundscape, music theory, spatial audio.
- **Narrative Lead**: World-building, plot, character arcs, dialogue.
- **QA Lead**: Test plans, bug triage, regression testing.
- **Release Manager**: CI/CD, store compliance, versioning.
- **Localization Lead**: Multi-language support, culturalization.

## Tier 3 — The Specialists (37 Agents)

### Engineering Specialists (10)
- **Gameplay Programmer**: Core loop, character controls, combat.
- **Engine Programmer**: Graphics, shders, physics, optimizations.
- **AI Programmer**: NPCs, behaviors, pathfinding.
- **Networking Programmer**: Multiplayer, sync, matchmaking.
- **Tools Programmer**: Editor extensions, build-pipe tools.
- **UI/UX Programmer**: Interface logic, menus, HUD.
- **Performance Analyst**: Frame-time, memory leak detection.
- **DevOps Engineer**: Automation, server-side infra.
- **Security Engineer**: Anti-cheat, data privacy.
- **Analytics Engineer**: Player telemetry, heatmaps.

### Design Specialists (10)
- **Systems Designer**: Math, spreadsheets, balancing logic.
- **Economy Designer**: Currencies, stores, monetization.
- **Level Designer**: Block-out, encounters, pacing.
- **World Builder**: Environment story-telling, lore integration.
- **Prototyper**: Fast iteration, gray-box testing.
- **Accessibility Specialist**: Color-blind, subtitles, remapping.
- **UX Designer**: User flow, wireframes, interaction.
- **Economy Designer**: (Redundant?) -> **Combat Designer**: Move-sets, balancing.
- **Difficulty Tuner**: Dynamic difficulty adjustment.
- **Live-Ops Designer**: Events, seasonal content.

### Art & Audio Specialists (10)
- **Technical Artist**: Shaders, rig-automation, optimization.
- **Environment Artist**: 3D assets, textures, lighting.
- **Character Artist**: Models, sculpting, anatomy.
- **Animator**: Kinematics, blend-trees, cutscenes.
- **VFX Artist**: Particles, post-processing, impact.
- **UI Artist**: Icons, menus, layout design.
- **Sound Designer**: SFX, foley, ambience.
- **Composer**: Music scores, adaptive soundtracks.
- **Voice Director**: Casting, recording supervision.
- **Asset Auditor**: Mesh-count, draw-call optimization.

### Production & QA Specialists (7)
- **QA Tester**: Manual testing, edge-case finding.
- **Automation Tester**: Scripted test suites.
- **Writer**: Scripts, quest-logs, character bibles.
- **Editor**: Script polish, consistency check.
- **Community Manager**: Marketing copy, player feedback loop.
- **Marketing Designer**: Store-front assets, trailers.
- **Producer-Support**: Burn-down charts, task-entry.
