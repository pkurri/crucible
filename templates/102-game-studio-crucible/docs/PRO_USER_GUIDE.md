## 2. Using the Revenue Optimizer (Pro Feature)

The **Revenue Optimizer** is your secret weapon for building profitable games.

### How to Trigger:
Run `/revenue-opt` during the Concept phase.

### What it Provides:
- **Free-to-Play (F2P) Hook Analysis**: Identify the best mechanics to gain users.
- **Pro Features Segmentation**: Carve out exclusive content for your "Pro" tier (e.g., expansion levels, VFX-only skins).

## 3. Incorporating into Any Project

You can bring the Gaming Studio into an existing project (Unity, Godot, etc.) in 3 steps:

1.  **Initialize**: Run `/studio-init` in the root of your existing project.
2.  **Audit**: Run `/game-audit` to let the 48 agents map your current system.
3.  **Modernize**: Use `/reverse-document` to generate a GDD and Revenue Plan for your existing code.
