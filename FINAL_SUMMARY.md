## 🚀 Getting Started

### Quick Start

```bash
# Clone repository
git clone https://github.com/pkurri/crucible.git
cd crucible

# Install dependencies
npm install

# Browse content
cat templates/CATALOG.md
cat skills/EXTENDED_CATALOG.md

# Use a template
cd templates/031-ai-code-reviewer
npm install
npm run dev
```

### Explore Skills

```bash
# View all skills
cat skills/EXTENDED_CATALOG.md

# View skill categories
ls skills/review-*
ls skills/tool-*
ls skills/workflow-*
```

### Explore Templates

```bash
# View all templates
cat templates/EXTENDED_CATALOG.md

# Browse by category
ls templates/031-* # AI/ML
```
