# 📜 Studio Commands (37 Skills)

The following skills are available as part of the **Gaming Studio** orchestrator (`gaming-studio`).

## 🔍 Review & Analysis (7)
- **/design-review**: Analyze the GDD for loop-holes, balance, or MDA alignment.
- **/code-review**: Domain-specific code audit (Art, Engine, Gameplay).
- **/balance-check**: Audit the economy (Free vs. Pro) and difficulty curves.
- **/asset-audit**: Check for mesh-count, draw-calls, and compression.
- **/scope-check**: Identify feature-creep and suggest scope-cuts.
- **/perf-profile**: Frame-time, memory, and load-time analysis (TURBO).
- **/tech-debt**: Map out legacy code and optimization needs.

## 🏗️ Production (5)
- **/sprint-plan**: Convert a PRD/GDD into 20+ atomic tasks (DISPATCH).
- **/milestone-review**: Audit major delivery gates (Alpha, Beta, Gold).
- **/estimate**: Complexity vs. Duration analysis for specific features.
- **/retrospective**: Continuous improvement for the studio workflow.
- **/bug-report**: Standardized issue tracking with fix-it loops (GLITCH).

## 🚀 Project Management (6)
- **/studio-init**: Detect project stage and initialize the 48-agent roster.
- **/project-stage-detect**: Analyze an existing repo for architectural patterns.
- **/reverse-document**: Generate a GDD from existing source code.
- **/gate-check**: Final validation before a major merge or release.
- **/map-systems**: Visualize the inter-agent dependencies.
- **/design-system**: Document UI/UX components and visual identity (ADD).

## 🌍 Release (5)
- **/release-checklist**: Pre-submission validation for Apple/Google (GATEWAY).
- **/launch-checklist**: Marketing, metadata, and deployment readiness.
- **/changelog**: Version history management.
- **/patch-notes**: User-facing update logs.
- **/hotfix**: Emergency fix workflow for production issues.

## ✍️ Creative (5)
- **/brainstorm**: Ideation and market-feasibility analysis (PULSE).
- **/playtest-report**: Analyze player feedback for UX improvements.
- **/prototype**: Rapid iteration and gray-box development.
- **/onboard**: Setup for new developers or engine specialists.
- **/localize**: Multi-language and culturalization audit.

## 🤝 Team Orchestration (9)
- **/team-combat**: Orchestrate Gameplay, AI, and Animation for combat.
- **/team-narrative**: Orchestrate Writers, World-builders, and Quest-designers.
- **/team-ui**: Orchestrate UI Artists, UX Designers, and UI Programmers.
- **/team-release**: Orchestrate QA, Release Manager, and DevOps.
- **/team-polish**: Orchestrate VFX, Audio, and Performance.
- **/team-audio**: Orchestrate Sound Designers, Composers, and Voice Directors.
- **/team-level**: Orchestrate Level Designers, Environment Artists, and LDs.
- **/team-revenue**: (New) Orchestrate Economy Designers and Product Managers.
- **/team-security**: (New) Orchestrate Security and Backend-Engineers.
